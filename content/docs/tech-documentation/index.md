## Пример 1: Документация MDM API

Mobile Device Management (MDM) API — API для управления мобильными устройствами в корпоративной среде. MDM позволяет ограничивать и контролировать функциональность устройств, обеспечивая безопасность и стандартизацию управления парком устройств.

В составе команды технических писателей я:

- Переводил исходники C++ документации с английского на русский при помощи системы автоматизированного перевода OmegaT.
- Использовал собственные разработки компании для парсинга исходников документации в формат Markdown.
- По своей инициативе перереботал структуру данного раздела документации. Сгруппировал страницы документации по модулям внутри раздела, благодаря чему пользователям стало проще искать информацию про функциональность API.
- Взаимодействовал с разработчиками MDM API и другими техническими писателями, чтобы получить фидбек по проделанной работе и провести ревью.

**[Подробная документация MDM →](https://developer.auroraos.ru/doc/software_development/reference/mdm)**
