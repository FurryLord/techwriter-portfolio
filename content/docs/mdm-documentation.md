## Пример 1: MDM API Documentation

Комплексная документация для Mobile Device Management (MDM) API операционной системы Аврора — фреймворка для управления мобильными устройствами в корпоративной среде. MDM позволяет ограничивать и контролировать функциональность устройств, обеспечивая безопасность и стандартизацию управления парком устройств.

**Что было задокументировано:**

- **Библиотеки и компоненты**: libsailfishpolicy для политик ограничений настроек и функций, libsailfishmdm для конкретных системных возможностей
- **API по подсистемам**: управление приложениями, сетевой стек, блокировка устройства, местоположение, логи, телефония, обновления
- **Разработка и интеграция**: требования к MDM-приложениям, интеграция с Systemd, политики безопасности, C++ API, QML привязки, D-Bus интерфейсы

**Результат:** Полная справочная документация MDM API, стандартизирующая подходы к управлению устройствами в корпоративной среде.

**[Подробная документация MDM →](https://developer.auroraos.ru/doc/software_development/reference/mdm)**

## Пример 2: Установка Аврора SDK для Linux

Подробная документация по установке и настройке Aurora SDK для операционной системы Linux. Руководство включает все этапы от подготовки рабочего места до запуска среды разработки.

**Что было задокументировано:**

- **Системные требования**: аппаратные требования, поддерживаемые ОС (Ubuntu, Альт), необходимое ПО (VirtualBox, Docker)
- **Процесс установки**: пошаговые инструкции по загрузке, запуску установщика, выбору компонентов
- **Конфигурация технологий виртуализации**: настройка VirtualBox и Docker, выбор подходящей технологии
- **Устранение неполадок**: известные проблемы и их решения (например, проблема с libxcb-xinerama.so.0 в Ubuntu 22.04)
- **Удаление SDK**: подготовка и процесс полной очистки системы от компонентов SDK

**Особенности документации:**

- **Многоуровневая структура**: от общих требований до конкретных команд
- **Визуальная поддержка**: скриншоты интерфейса установщика на каждом этапе
- **Условная логика**: рекомендации по выбору технологии в зависимости от условий (доступ к интернету)
- **Кросс-ссылки**: ссылки на смежную документацию (Docker, офлайн-установка)

**Результат:** Исчерпывающее руководство, позволяющее разработчикам успешно установить и настроить Aurora SDK в Linux-среде.

**[Документация по установке SDK →](https://developer.auroraos.ru/doc/sdk/app_development/setup/setup_linux)**
